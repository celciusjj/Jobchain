## ¿Cómo ejecutar en Android?

1. Clona este repositorio:

   ```bash
   git clone https://github.com/celciusjj/Jobchain.git
   ```

2. Instala las dependencias:

   ```bash
   npm install
   ```

3. Inicia el proyecto:

   ```bash
   npm start
   ```

4. Ejecuta la app en Android:
   ```bash
   npm run android
   ```
